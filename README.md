## Tech Stack

### Frontend
- React.js with Vite
- React Router for navigation
- Tailwind CSS for styling
- Axios for API requests
- Leaflet for maps

### Backend
- Node.js & Express.js
- Supabase (PostgreSQL) for database
- JWT for authentication
- bcrypt for password hashing
- PostGIS for geospatial data
